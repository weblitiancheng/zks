下载说明

1. 配备Intel Xeon E5-v4系列 CPU的机型不支持Windows Server 2008 R2操作系统；


2. 下载的文件为exe格式的，请直接运行，根据向导提示解压到指定目录，并前往该目录获取驱动文件；


3. 请根据服务器顶部的服务器产品标签所注配置（板载SATA 或 SAS RAID）下载对应阵列卡驱动；


4. 板载SATA驱动中有数个压缩包（"windows7_2008R2_x64.zip"适用于Windows Server 2008 R2操作系统，"windows8_ws2012_x64.zip"适用于Windows Server 2012操作系统，"windows8_1_ws2012R2_x64.zip"适用于Windows Server 2012 R2操作系统），安装操作系统前需先将压缩包解压到U盘上，并在安装系统时手动加载驱动，浏览指定该文件夹，加载对应驱动；


5. SAS RAID驱动中有数个压缩包（"win7_x64.zip"适用于Windows Server 2008 R2操作系统，"win8_x64.zip"适用于Windows Server 2012操作系统，"win8.1_x64.zip"适用于Windows Server 2012 R2操作系统，"win10_x64.zip"适用于Windows Server 2016操作系统），安装Windows Server 2008 R2及2012操作系统前需先将压缩包解压到U盘上，并在安装系统时手动加载驱动，浏览指定该文件夹，加载对应驱动（Windows Server 2012 R2及2016系统已自带驱动）；


6. 在Windows Serve 2008 R2操作系统安装芯片组驱动前需要启用系统的.NET Framework 3.5.1功能；


7. 解压缩网卡驱动文件后，运行"Autorun.exe"，选择“安装驱动程序和软件”，然后按照安装向导操作；


8. 解压缩显卡驱动文件后，需要在Windows设备管理中手动更新“显示适配器”的驱动程序；


9. USB3.0驱动仅适用于Windows Server 2008 R2操作系统，Windows Server 2012及以上系统已自带；


10. 配备有安全芯片(TCM)的机型需要安装TCM驱动，未配备安全芯片的机型不需要安装。
